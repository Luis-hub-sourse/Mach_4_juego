El proyecto de persistecia tendra definidas las clases de repositorio para acceder a los datos de las tarjetas almacenadas en un MongoDB.

El repositorio debera realizar las siguientes acciones:

- Obtener todas las tarjetas disponibles.
- Obtener todas tarjetas de una categoria especifica.

Se debe devolver una lista de objetos de tipo "Tarjeta" al realizar las consultas.

No adivines, si hay dudas preguntar.
